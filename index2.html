Question-1: What are the different ways to select an element in the DOM?

Selecting by ID: document.getElementById('elementId')

Selecting by Class Name:document.getElementsByClassName('className')

Selecting by Tag Name:document.getElementsByTagName('tagName')

Query Selector:document.querySelector('cssSelector')

Query Selector All:document.querySelectorAll('cssSelector')

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events from multiple child elements using event bubbling, improving performance and handling dynamic elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where, when you click on an element (like a button), the event first happens on that element, and then it "bubbles" up to its parent elements (like a div or body), triggering events on them as well.
